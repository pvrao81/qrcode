# Custom QR Code Generator

A simple Python-based QR Code Generator that allows users to create QR codes with **custom colors** and an **optional logo** in the center. The app provides a live preview and lets users download the generated QR code as a PNG file.

## Features
- Generate QR codes from text or URLs
- Choose custom QR code and background colors
- Add an optional transparent logo to the center
- Preview the QR code before downloading
- Download QR codes as PNG images

## Technologies Used
- Python
- Gradio (web interface)
- qrcode
- Pillow (image processing)
